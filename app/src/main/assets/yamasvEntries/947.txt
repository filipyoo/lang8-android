
Using 〜ばいい and 〜たらいい

They are used to recommend something.

おいしい寿司が食べたかったら、この店に 行けばいい / 行ったらいい。
- If you want to have delicious sushi, you should visit this restaurant.

日本語のスラングに興味があるなら、この本を 読めばいい / 読んだらいい。
- If you are interested in Japanese slang, you should read this book.

スポーティで贅沢な車が欲しいなら、BMWにすればいい / BMWにしたらいい。
- If you want a sporty and luxury car, you should buy a BMW.


They are also used to give permission.

日本語の会話の練習をしたい時は、いつでも私に 電話すればいい / 電話したらいい。
- You can call me anytime you want to practice Japanese conversation.

お腹が空いてるなら、僕のサンドイッチを 食べればいい / 食べたらいい。
- You can eat my sandwich if you are hungry.

車が必要なら、僕の車を 使えばいい / 使ったらいい。
- You can use my car if you need one.
