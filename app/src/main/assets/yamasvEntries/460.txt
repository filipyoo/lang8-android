
The Usage of 〜もある/いる 

You can use 〜もある/いる to say "some 〜."

多くの人がスマートフォンを使ってるが、使ってない人もいる。
Many people use a smart phone, but some people don't.

この中のほとんどのおもちゃは日本製ですが、アメリカ製もあります。
Most of these toys are made in Japan, but some are made in the USA.

これらの薬の中には、副作用をひきおこすものもある。
Some of these drugs can cause side effects.

このクラスの生徒の中には、流暢に日本語を話せる人もいます。
Some students in this class can speak Japanese fluently.


If you say "このクラスの生徒の中には、流暢に日本語を話せる人 が います," it sounds as if only one student can speak Japanese fluently. The same holds true for the rest of all sentences above.
