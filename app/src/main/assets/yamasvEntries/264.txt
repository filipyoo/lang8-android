
The Omission of Particles.

You may have experience that you didn't hear particles when you heard Japanese people's conversations. It might be not because that you failed to hear the particles but because they intentionally omitted the particles. We often omit particles in casual conversation. I'd like to explain when and what particles can be omitted and when and what particles can't.

1. は
は which is used as a topic marker is often omitted in casual conversation.

昨日は、どこに行ったの？ / 昨日、どこに行ったの？
Where did you go yesterday?

田中さんは、結婚するんだって？ / 田中さん、結婚するんだって？ / 
Is it true that Tanaka-san is going to get married?

このケーキは、美味しいね / このケーキ、美味しいね。
This cake is delicious, isn't it?

The second sentences in each example sound more natural. Many Japanese people would omit は in these examples.

2. は
You can't omit は which is used to show contrast.

魚は好きだけど、肉は嫌い / 魚好きだけど、肉嫌い。
- I like fish. However I don't like meat.

日本には行ったことあるけど、米国には行ったことない / 日本に行ったことあるけど、米国に行ったことない。
- I have been to Japan before. However I haven't been to America.

The second sentences in each example sound weird.

3. が

田中さんが来たら、教えて。/ 田中さん、来たら、教えて。
Please tell me when Tanaka-san will come.

この仕事が終わったら、飲みに行こう / この仕事終わったら、飲みに行こう。
Let's go drink after we finish this job.

All of the four sentences sound natural, I would omit が in these examples.

4. を

昨日の試合を見た？ / 昨日の試合、見た？
Did you see the game yesterday?

あのケーキを食べてみたい / あのケーキ、食べてみたい。
I'd like to try that cake.

値段を見た時、びっくりした / 値段見た時、びっくりした。
I was surprised when I saw the price.

In case of the examples above, omitting を sounds more natural. Especially, "昨日の試合 を 見た？" sound a little bit unnatural.

5. に
You can omit に when it refers to the destination, but you can't omit に when it refers to the places where the things exist.

o 日本に着いたら電話してね / o 日本着いたら電話してね。
Call me when you will arrive in Japan.

o 公園に犬がいたよ / x 公園犬がいたよ。
There was a dog in that park.


6. で  と  から
You can't omit で, と and から as far as I know.
