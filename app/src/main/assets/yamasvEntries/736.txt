
Using ようになる

ようになる is used when you have started some repeated or continued action that you haven't done before.

最近、毎日日本語を勉強するようになった。
- Lately, I study Japanese every day.

子供の時にアニメを見てから、日本に興味を持つようになった。
- I've been interested in Japan since I watched anime when I was a child.
If you say "興味を持った," it sounds as if you was interested in only for a short time.

タバコを吸うようになった。
- I've started smoking.
You can't say タバコを止めるようになった because 止める is not a repeated or continued action, but you can say タバコを吸わないようになった to mean "I quit smoking."


It's also used when you've got an ability or opportunity of something.

赤ちゃんが簡単な言葉を話せるようになった。
- Our baby starts saying simple words.

最近、辞書なしで漫画が読めるようになった。
- Lately, I can read manga without a dictionary.

目の手術をしてから、はっきりものが見えるようになった。
- I can see anything clearly after I had an eye surgery.

就労ビザがとれたので、日本で働けるようになった。
- I've got a working visa, so I can work in Japan. 



You can also use it for the future.

この漫画を見たら日本に興味を持つようになるだろう。
- I'm sure you will have been interested in Japan if you read this manga.

赤ちゃんはもうすぐ歩けるようになるだろう。
- Our baby will start walking soon.
