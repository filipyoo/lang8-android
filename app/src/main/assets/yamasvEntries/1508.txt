
Using 〜とうれしい

〜とうれしい or 〜たらうれしい means that "I'd be happy if 〜." You can use it to ask someone to do something indirectly.

手伝ってくれるとうれしい / 手伝ってくれたらうれしい。
- I'd be happy if you could help me.
It implies that "You don't have to if you are busy."

週末どこか行きたいところはある？
ディズニーランドに行けるとうれしい / 行けたらうれしい。
- Do you have anywhere you want to go this weekend?
- I'd be happy if we could go to Disney Land. 

私の誕生日パーティに来てくれるとうれしい / 来てくれたらうれしい。 
- I'd be happy if you would come to my birthday party.

私の誕生日パーティに来てくれたらうれしかったのに。 
- I would have been happy if you had come to my birthday party.
In this case, 来てくれるとうれしかった sounds a little unnatural to me.
