
Using うち

うち can be used to refer to a time.

一生懸命練習すれば、そのうち日本語を流暢に話せるようになる。
- If you practice hard, eventually you will speak Japanese fluently.

近いうちに日本にまた来たい。
- I'd like to visit Japan again before long.
(The verb of this Japanese sentence depends on where you are now. If you haven't left Japan yet, you should say 来たい.  If you have already returned to your country, you should say 行きたい.)

明るいうちに家に帰って来なさい。
暗くならないうちに家に帰って来なさい。
- You should return home before dark.

日本にいるうちに、たくさんのお寺を訪れるつもりです。
- I'm going to visit a lot of temples while I'm in Japan.
