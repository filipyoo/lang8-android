
I said that you can express conditional by using と yesterday.

These are examples of conditional statement using と.

日本に行く と、富士山を見ることができる - If you go to Japan, you can see Mt. Fuji. 
日本に行く と、忍者に会える - If you go to Japan, you can meet a Ninja (It's not true.)


However, you can't always express conditional by using と. You can't use と when you want to express your desires or plans. 


x 日本に行く と、富士山を見たい

You need to use たら or なら in this case.

○ 日本に行っ たら、富士山を見たい  - If I go to Japan, I want to see Mt. Fuji. 
○ 日本に行く なら、富士山を見たい 


Furthermore, You can't use と when you ask someone to do something or recommend something. You need to use たら or なら　in this case.

x 日本に行く と、田中さんに会ってください 
○ 日本に行っ たら、田中さんに会ってください - If you go to Japan, please see Mr. Tanaka.
○ 日本に行く なら、田中さんに会ってください

x 日本に行く と、寿司を食べるべきです  
○ 日本に行っ たら、寿司を食べるべきです - If you go to Japan, you should eat sushi by all means.
○ 日本に行く なら、寿司を食べるべきです


I will explain the difference between たら and なら in another entry.
