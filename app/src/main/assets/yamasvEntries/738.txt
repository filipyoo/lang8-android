
Using ている for the future

ている is mainly used to describe what you are doing now or you do repeatedly.

今、ゴルフをしている。
- I'm playing golf now.

私は英語を教えている。
- I teach English.


ている can also be used to describe a future event that you have arranged to do. It may be similar to English.

今日の午後はゴルフをしている。
- I'm playing golf this afternoon.

明日は一日中働いているから、君とは会えない。
- I can't see you because I'm working all day tomorrow.


However, you can't say ている if the action ends in a short time. You should use the dictionary form in this case.

午後、彼に電話をかける。
- I'm phoning him this afternoon.
If you say 午後は電話をかけている, it sounds as if you are talking with him in a whole this afternoon.

明日、新しいiPhoneを買う。
- I'm buying a new iPhone tomorrow.
